📥 Fuentes de datos — Estado y actualización

🌐 World Bank API — Carga en vivo automática

La app puede sustituir los datos WGI y WDI embebidos por datos reales del Banco Mundial. Sin clave, totalmente gratuito.

  1. Haz clic en "🌐 Cargar WB Live" en el header o abajo. La app llama a api.worldbank.org/v2/ y descarga los 7 indicadores WGI + 7 indicadores WDI para todos los países
  2. El proceso tarda ~30–60 segundos (paginación automática). El mapa se actualiza en tiempo real según llegan los datos
  3. El indicador "Fuente WB" en el panel superior cambia de Embebido a Live XXXX cuando termina
  4. Si falla la conexión, los datos embebidos siguen activos como fallback

🌍 REST Countries API — Datos de país enriquecidos

Carga capital, idiomas, moneda, región y banderas de cada país desde restcountries.com/v3.1/all. Sin clave.

  1. Haz clic en "↺ Cargar REST Countries"
  2. Se descargan ~250 países en una sola petición (JSON ~2MB)
  3. Los datos aparecen en el modal de cada país (capital, idioma, moneda…)

🏛 WGI — Actualización manual (CSV)

Fuente: worldbank.org → Worldwide Governance Indicators

  1. Descarga el archivo Excel desde la sección "Download the data"
  2. Pestaña "WGIData" → filtra el año más reciente → columnas Code, PV.EST, GE.EST, CC.EST, RL.EST, VA.EST, RQ.EST
  3. Copia como CSV en el formato indicado y pega abajo

Formato: ISO2,PV,GE,CC,RL,VA,RQ

📊 WDI — Actualización manual (CSV)

Fuente: databank.worldbank.org → World Development Indicators

  1. Selecciona Country: All · Series: NY.GDP.PCAP.PP.KD, EG.ELC.ACCS.ZS, EN.ATM.CO2E.PC, AG.LND.FRST.ZS, SL.UEM.TOTL.ZS, SE.ADT.LITR.ZS, SI.POV.NAHC
  2. Time: año más reciente · Download → CSV

Formato: ISO2,GDP,EL,CO2,FR,UN,LIT,POV

🔍 CPI · 🗽 Freedom House — Manual

CPI: transparency.org/en/cpi → "Download the data" → Excel → columnas ISO2 + CPI score

Freedom House: freedomhouse.org → "Download data" → Excel → columna CL (1 libre – 7 no libre)

⚡ GDELT — Snapshot manual

Formato: ISO2,instability (0–0.20, mayor = más inestable)

Índice de Estabilidad GDELT (0–100)

Estabilidad = max(0, 100 − instabilidad × 500). Rango: 0 (crisis) → 100 (máxima calma).

Fórmula CountryScore

CountryScore = Σ(variable_i × peso_i) / Σ(pesos disponibles) Penalización GDELT: instabilidad > 0.08 → × 0.40 (guerra abierta) instabilidad > 0.04 → × 0.65 (conflicto activo) instabilidad > 0.02 → × 0.85 (tensión)
